# OpenTelemetry Astronomy Shop Observability Lab

## Project Overview
This project demonstrates an end-to-end observability stack using the OpenTelemetry Astronomy Shop microservices demo.

The system is deployed locally on a VM using Docker Compose and includes:

- OpenTelemetry Collector
- Distributed tracing
- Metrics collection
- Grafana dashboards
- Prometheus
- Jaeger (or Tempo)

## Goals

- Deploy a microservice architecture locally
- Instrument services using OpenTelemetry
- Configure the OpenTelemetry Collector
- Visualize traces and metrics
- Build a portfolio-ready observability lab
